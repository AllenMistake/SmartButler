# SmartButler 智能管家

## 项目介绍

来自于慕课网的智能管家App开发，由于原版在2016年左右，
所以对于Android6.0之后的系统版本不是很兼容，所以，在
这个的基础上对
高版本的Android系统做一些适配

## 使用到的开源库

- Rxvolley
- 腾讯Bugly
- bmob后端云
- 数据聚合

## 对于原版程序的改进

- 适配由于Android6.0+运行时权限造成对于路径访问的Bug
